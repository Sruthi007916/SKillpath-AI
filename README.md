# SkillPath AI

## AI-Powered Personalized Learning Path Recommendation System

SkillPath AI is a full-stack web application designed to evaluate users' technical skills through an interactive assessment and provide personalized career recommendations with a structured learning roadmap.

The platform helps students and aspiring professionals identify suitable career paths based on their assessment performance and offers skill recommendations to bridge knowledge gaps.

## Installation

### Clone the Repository

```bash
git clone https://github.com/Sruthi007916/SkillPath-AI.git
```

### Navigate to the Project Folder

```bash
cd SkillPath-AI
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure MySQL

- Create the database.
- Import the `skillpath.sql` file.
- Update the database credentials in `backend/db.py`.

### Run the Application

```bash
python backend/app.py
```
